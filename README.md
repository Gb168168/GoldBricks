# GoldBricks 員工請假系統（免登入）

目前系統啟動時會先進入登入頁，
可選擇登入使用者與「桌機版 / 手機版」顯示模式（資料完全共用）。

## 使用方式

1. 開啟系統後，先選擇登入使用者
2. 在登入頁選擇「桌機版」或「手機版」
3. 兩個版本共用同一份資料，只是顯示比例與版面不同
4. 進入系統後可在右上角快速切換版本
5. 若目前登入者不是管理員，可點選右上角「設為最高權限」取得設定與審核權限

## 管理員如何新增員工

1. 進入「員工管理」→「新增員工」
2. 填寫員工基本資料
3. 儲存後員工資料即加入系統

## Firebase 在本專案中的角色

- 專案仍保留 Firebase 專案設定（`firebaseConfig`），方便後續擴充。
- 目前流程不依賴 Firebase 驗證。
