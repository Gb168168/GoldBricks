# GoldBricks 員工請假系統（帳號密碼登入）

目前系統登入方式已改為：**員工帳號（username）+ 系統密碼（password）**，
**不使用 Email 當登入帳號**。

## 登入規則

1. 登入畫面輸入「帳號」與「密碼」
2. 系統會比對內建/本機資料中的 `username`、`password`
3. 驗證成功即可登入

> 例如預設管理員帳號為 `001`，預設密碼也是 `001`（可在員工管理中修改）。

## 管理員如何新增員工

1. 以管理員登入
2. 進入「員工管理」→「新增員工」
3. 設定員工代號、登入帳號、登入密碼（至少 4 碼）
4. 儲存後，員工可使用該帳號密碼登入

## 編輯員工密碼

- 編輯員工時，若「登入密碼」留空，會保留原密碼。
- 若有輸入新密碼，會更新為新密碼。

## Firebase 在本專案中的角色

- 專案仍保留 Firebase 專案設定（`firebaseConfig`），方便後續擴充。
- 目前登入流程不依賴 Firebase Email/Password。

## 常見問題

### 我不要用信箱當帳號，現在可以嗎？
可以。現在登入只看 `username + password`。

### 忘記密碼怎麼辦？
請管理員到員工管理中編輯該員工，輸入新密碼後儲存。
