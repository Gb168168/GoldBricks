# GoldBricks 員工請假系統（免登入）

目前系統已移除登入畫面，開啟後會直接進入系統，
並預設使用第一位員工資料（預設為管理員）。

## 使用方式

1. 直接開啟系統即可使用，不需輸入帳號密碼
2. 若資料中有多位員工，系統會記住上次使用者
3. 點選右上角「回到預設使用者」可快速切回預設帳號
4. 若目前登入者不是管理員，可點選右上角「設為最高權限」取得設定與審核權限

## 管理員如何新增員工

1. 進入「員工管理」→「新增員工」
2. 填寫員工基本資料
3. 儲存後員工資料即加入系統

## Firebase 在本專案中的角色

- 專案仍保留 Firebase 專案設定（`firebaseConfig`），方便後續擴充。
- 目前流程不依賴 Firebase 驗證。
